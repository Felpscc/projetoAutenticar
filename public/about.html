Nossa Missão

Nossa missão é fornecer uma plataforma intuitiva e eficaz para estudantes de todas as áreas organizarem seus estudos, definirem metas e acompanharem seu progresso. Acreditamos que a organização é a chave para o sucesso acadêmico e pessoal.

Tecnologias Utilizadas

Esta plataforma foi desenvolvida utilizando:

  • Node.js com Express para o backend
  • SQLite como banco de dados para simplicidade e portabilidade
  • HTML5, CSS3 e JavaScript para o frontend
  • bcryptjs para segurança de senhas
  • express-session para gerenciamento de sessão
